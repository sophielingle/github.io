CSCE 242: Website Applications

Website Applications is a course designed to teach students how to develop applications for different situations. Not only are these websites created, but designed with efficiency in mind for the user. Languages such as HTML, CSS, and JavaScript will be used to complete assignments.

Assignments

Assignment 1 - Basic HTML

This assignment taught us how to build a basic web page using html. I built a website describing the pros of vinyl records.

Assignment 2 - Basic CSS

This assignment taught us how to design a web page using CSS. I created a web page about my favorite dog breed and added colors and backgrounds.

Assignment 3 - Page Layout

This assignment taught us how to format and align a web page using CSS flex.
